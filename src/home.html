Edcho-Server

This web server will echo back the JSON you send to it (when called correctly) along with the generated data (Heroes information) from the simulated downstream system.

You can also force the server to return errors by adding key data to the JSON document as input to a POST call.

You can call the API using curl or some other command to make a POST http call.

curl -i \
	-H "Content-Type: application/json" \
	-H "Accept: application/json" \
	-H "X-HTTP-METHOD-Override: PUT" \
	-X POST \
	-d '{"username":"xyz","password":"xyz"}' \
	localhost:3000/api/v1/echo

Here is the same command on one line:

curl -i -H "Content-Type: application/json" -H "Accept: application/json" -H "X-HTTP-METHOD-Override: PUT" -X POST -d '{"username":"xyz","password":"xyz"}' localhost:3000/api/v1/echo
